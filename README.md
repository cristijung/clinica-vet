# Tutorial Node, Express, Prisma e React (Next.js)

## Objetivo: 
Criar uma API robusta para uma clínica veterinária com uma estrutura que separe bem as responsabilidades. Vamos construir uma REST API simplificada, mas com padrões profissionais, utilizando Node.js, Express, Prisma (ORM) e TypeScript.
### O tutorial é uma prática para criar uma API de uma clínica veterinária. Ele abrange:
- ***Setup do Backend:*** Instalação de Node.js, Express e TypeScript.
- ***Banco de Dados:*** Configuração do Prisma ORM com SQLite, criação de modelos (Pet) e execução de migrations.
- ***Frontend:*** Criação de um app Next.js, uso de Axios para consumo da API (GET, POST, DELETE) e estilização com Tailwind.
