# Projeto API-REST
## Node.js + Express + Prisma + SQLite
### Projeto Clínica Veterinária

#### Instalações:
- Prisma: 
    - `npm install @prisma/client`
    - `npm install prisma --save-dev`
    - Inilialição do Prisma: `npx prisma init --datasource-provider sqlite`
Obs: Vamos usar o ***SQLite*** para este tutorial. Ele criará um arquivo local `.db` sem precisar instalar um servidor de banco de dados.
